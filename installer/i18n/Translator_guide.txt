VDJC Installer Translations

This folder contains the custom installer strings used by Inno Setup.
Edit only installer/i18n/VDJC_CustomMessages.isl.

How it works
- The main script (installer/VDJC_Setup.iss) includes this file.
- Each entry uses the format: <language>.<Key>=<Text>
- Keep keys exactly the same across languages.
- Use #13#10 for line breaks inside long texts.

What you should translate
- InfoText (long info page text)
- RequirementsTitle, RequirementsSubTitle, RequirementsDesc, RequirementsCheckbox
- ControllersTitle, ControllersSubTitle, ControllersDesc, ControllersLabel
- ScriptTitle, ScriptSubTitle, ScriptDesc, ScriptLabel
- ErrControllers, ErrRequirements, ErrScript

Do not change
- The keys (left side of =)
- #13#10 sequences
- File/variable names like VDJC, MIXXX, UpdatePresetsAndStartMixxx.ps1

ASCII note
- This repository currently uses ASCII-only text in translations.
- For non-Latin languages, use English for now or provide a transliteration.

Quick check
- After editing, run the installer build to verify there are no syntax errors.
