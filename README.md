# VDJC-Scripts-Installer
Installer for MIXXX scripts and utilities, required to make the Virtual Controller work properly.

## Initial public release

This is the first public release of the **VDJC Scripts Installer for MIXXX**.

### Included
- VDJC controller mapping files for MIXXX
- VDJC JavaScript controller script
- Helper PowerShell script for automatic effect and preset synchronization
- Windows installer with integrated uninstaller

### Notes
- MIXXX must be installed before running the installer
- The PowerShell script is optional but recommended to keep VDJC synchronized with MIXXX
- No MIXXX user configuration files are modified or removed

This release provides a clean and automated way to set up VDJC on Windows systems.
